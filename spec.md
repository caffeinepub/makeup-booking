# Specification

## Summary
**Goal:** Restore the Naaz Studio application to the state it was at version 17, reverting all changes made after that version.

**Planned changes:**
- Revert all frontend changes introduced after version 17
- Revert all backend changes introduced after version 17

**User-visible outcome:** The application behaves and appears exactly as it did at version 17, with no changes from later versions present.
